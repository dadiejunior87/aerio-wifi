# ⚡ AERIO | Système de Gestion Wi-Fi Pro

![Version](https://img.shields.io/badge/Version-1.0_Alpha-00C2FF?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Stable-00F5A0?style=for-the-badge)

**AERIO** est une solution légère et moderne (Web-App) conçue pour les partenaires de réseaux Wi-Fi. Elle permet de gérer la création de codes, la vente rapide avec impression de tickets et le suivi comptable en temps réel, le tout sans avoir besoin d'un serveur complexe.

---

## 🚀 Fonctionnalités Clés

* **Tableau de Bord Intelligent :** Vue globale sur les ventes et l'état du stock.
* **Gestion des Tarifs :** Configurez vos offres (ex: 1h, 24h, 1 Mois) en quelques clics.
* **Terminal de Vente :** Génération de codes et interface d'impression optimisée pour les tickets de caisse.
* **Historique des Recettes :** Journal complet des ventes avec export CSV pour la comptabilité.
* **Interface Multilingue :** Disponible en Français, Anglais, Portugais et Espagnol.
* **Mode Offline :** Fonctionne entièrement via le stockage local (LocalStorage) pour une rapidité maximale.
