⚙️ ÉTAPES DE CONFIGURATION RAPIDE

1. Liaison Nœud

Copiez le script généré dans "Ajouter Wifi" et collez-le dans le Terminal de votre MikroTik via Winbox.

2. Synchro Profils

Le système injecte automatiquement les débits (1h, 24h, etc.) dans votre Hotspot User Profiles.

3. Prêt à Vendre

Une fois le voyant au vert, vos tickets générés sont immédiatement fonctionnels sur votre réseau.

⚡ OPTIMISATION : ACHAT SANS DATA

Permettez à vos clients de payer même sans forfait. Dans MikroTik, allez dans IP > Hotspot > Walled Garden et ajoutez :

Dst. Host: *aerio-pay.com* | Action: allow

Note : Ajoutez aussi les domaines Orange/MTN Money pour valider les transactions.
